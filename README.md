# Welcome Onboarding

This Pelican plugin replaces the built-in account-created email with a configurable onboarding email.

## Features

- Custom subject, heading, and intro copy
- Setup-account button that keeps Pelican's reset-password flow
- Optional getting-started, Discord, community, and support links
- First-login instructions with one step per line
- Extra custom links from the plugin settings screen

## How To Use

1. Open the admin panel.
2. Go to `Plugins`.
3. Make sure `Welcome Onboarding` is enabled.
4. Open the plugin `Settings` action.
5. Save your welcome content and links.
